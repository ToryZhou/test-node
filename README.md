Chrome浏览器密码框自动填充的bug
了解决Chrome浏览器密码框自动填充的问题，经过查找资料，此问题已经解决了，就是在原有的input标签前面再加个input标签，并给此input标签设置相应的属性
<!-- 额外增加的input -->
<input type="password" style="width:0;height:0;float:left;visibility:hidden"/>
<!-- 原先的input -->
<input type="password"/>
